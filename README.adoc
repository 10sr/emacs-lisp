Public Repository
-----------------

Some packages in this repository can be installed with `package.el`.
To enable this repository, add lines to your init file:

----
(setq package-archives
      `(,@package-archives
        ("10sr-el" . "http://10sr.github.io/emacs-lisp/elpa/")))
----
